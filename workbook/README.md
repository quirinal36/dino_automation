# Dino Game Automation Workbook

> **미션**: "비디오 게임을 대신 플레이 해주는 AI 만들기"

이 워크북은 **제1원칙 사고방식(First Principles Thinking)**을 훈련하기 위한 수업 과정의 일부입니다. 아무것도 없는 상태에서 Chrome Dino 게임을 자동으로 플레이하는 봇을 만들어 봅니다.

---

## 대상 독자

- Python 기본 문법을 아는 초보자
- 라이브러리 사용 경험이 없어도 괜찮습니다
- 스스로 생각하고 코드를 작성할 의지가 있는 분

---

## 최종 결과물

이 워크북을 완료하면 다음을 만들 수 있습니다:

```
dino_automation/
├── calibrate.py      # ROI 캘리브레이션 도구
├── main.py           # 게임 자동화 엔진
├── roi_config.json   # 설정 파일 (자동 생성)
└── debug_captures/   # 디버그 스크린샷 (자동 생성)
```

---

## 사용하는 기술 스택

| 라이브러리 | 역할 | 배우는 챕터 |
|-----------|------|------------|
| **Pillow** | 화면 캡처 | 3장 |
| **NumPy** | 이미지를 숫자 배열로 처리 | 4-5장, 7장 |
| **OpenCV** | 이미지 처리, UI 구현 | 6장, 10장 |
| **PyAutoGUI** | 키보드 자동화 | 8장 |
| **JSON** | 설정 파일 저장/로드 | 9장 |

---

## 핵심 용어

| 용어 | 의미 | 배우는 챕터 |
|------|------|------------|
| **ROI** (Region of Interest) | 관심 영역 - 장애물을 감시할 화면의 특정 사각형 영역 | 2장, 5장 |
| **Calibration** (캘리브레이션) | 보정 - 각 사용자의 화면에 맞게 ROI 좌표를 설정하는 과정 | 2장, 10장 |
