## 우리가 만들 것

Chrome 브라우저에 숨겨진 공룡 게임을 아시나요? 인터넷이 끊겼을 때 나타나는 그 게임입니다.

우리는 이 게임을 **자동으로 플레이하는 봇**을 만들 것입니다.

### 봇이 하는 일

1. 화면을 "본다" (스크린샷 캡처)
2. 장애물이 있는지 "판단한다" (이미지 분석)
3. 필요하면 "점프한다" (키보드 입력)

### 최종 프로젝트 구조

```
dino_automation/
├── calibrate.py      # 화면 영역을 설정하는 도구
├── main.py           # 게임을 자동으로 플레이하는 메인 프로그램
├── roi_config.json   # 설정 파일 (프로그램이 자동 생성)
├── requirements.txt  # 필요한 패키지 목록
└── debug_captures/   # 디버그용 스크린샷 (프로그램이 자동 생성)
```

## 사용할 도구들

이 프로젝트에서는 5가지 Python 라이브러리를 사용합니다:

| 라이브러리 | 역할 | 비유 |
|-----------|------|------|
| **Pillow** | 화면 캡처 | 봇의 "눈" |
| **NumPy** | 숫자 배열 처리 | 봇의 "시각 피질" |
| **OpenCV** | 이미지 처리 | 봇의 "이미지 분석기" |
| **PyAutoGUI** | 키보드 입력 | 봇의 "손가락" |
| **JSON** | 설정 저장 | 봇의 "메모장" |

## 환경 설정

### Step 1: 프로젝트 폴더 만들기

원하는 위치에 `dino_automation` 폴더를 만드세요.

```
예: C:\Users\사용자이름\Documents\dino_automation
```

### Step 2: 가상 환경 만들기

터미널(명령 프롬프트)을 열고 프로젝트 폴더로 이동합니다.

```bash
cd C:\Users\사용자이름\Documents\dino_automation
```

가상 환경을 만듭니다:

```bash
python -m venv venv
```

### Step 3: 가상 환경 활성화

**Windows:**
```bash
venv\Scripts\activate
```

**Mac/Linux:**
```bash
source venv/bin/activate
```

활성화되면 터미널 앞에 `(venv)`가 표시됩니다.

### Step 4: requirements.txt 파일 만들기

프로젝트 폴더에 `requirements.txt` 파일을 만들고 다음 내용을 입력하세요:

```
opencv-python>=4.8.0
numpy>=1.24.0
pyautogui>=0.9.54
Pillow>=10.0.0
```

### Step 5: 패키지 설치

```bash
pip install -r requirements.txt
```

### Step 6: 설치 확인

다음 명령어로 모든 패키지가 설치되었는지 확인하세요:

```bash
python -c "import cv2, numpy, pyautogui, PIL; print('모든 라이브러리가 설치되었습니다!')"
```

"모든 라이브러리가 설치되었습니다!"가 출력되면 성공입니다.

## Chrome Dino 게임 실행하기

### 방법 1: 오프라인 모드

1. Chrome 브라우저를 엽니다
2. 인터넷 연결을 끊습니다
3. 아무 웹사이트에 접속을 시도합니다
4. 공룡이 나타납니다!

### 방법 2: 직접 접속 (권장)

1. Chrome 브라우저를 엽니다
2. 주소창에 `chrome://dino` 입력
3. Enter를 누릅니다
4. 스페이스바를 눌러 게임을 시작합니다
